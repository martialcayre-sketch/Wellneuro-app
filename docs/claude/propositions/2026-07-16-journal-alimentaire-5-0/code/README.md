# Code de référence

Ces fichiers définissent les contrats cibles et une politique d’observation hybride. Ils ne constituent pas une implémentation prête à fusionner.

Points à adapter dans le dépôt :

- conventions d’identifiants ;
- dates relatives `DAY_1`…`DAY_21` ;
- types communs `SourceRef` ;
- versions du corpus ;
- frontières C2A/JA/C5 ;
- règles de couverture validées.
