# Expérience patient — Ma spirale alimentaire

## 1. Direction d’expérience

L’expérience patient s’inscrit dans l’univers **Forêt & cuivre** : ivoire, vert forêt, sauge et cuivre. Elle doit évoquer la croissance et la continuité, sans devenir ludique ou infantilisante.

Le patient ne voit pas un tableau de performance. Il voit :

- où il en est dans la phase ;
- pourquoi une trace est utile aujourd’hui ;
- son action principale ;
- la façon la plus simple de renseigner ;
- la possibilité de faire moins lors d’une semaine difficile ;
- une réflexion factuelle en fin de semaine.

## 2. Accueil patient

```text
Ma spirale alimentaire
Phase 1 · jour 8 sur 21

Pourquoi aujourd’hui ?
Vous testez un petit-déjeuner contenant une source de protéines.
Une trace sur votre petit-déjeuner suffit aujourd’hui.

[Ajouter ma trace]
[Utiliser mon plan minimal]
```

Le compteur de jours informe, mais n’évalue pas. Pas d’anneau de performance, de flamme, de série ou de pourcentage de réussite.

## 3. Saisie en quatre niveaux

### Niveau 0 — rien à saisir

Si aucune observation n’est utile ce jour-là, l’application reste silencieuse.

### Niveau 1 — trace instantanée

Un clic depuis un raccourci :

- « comme d’habitude » ;
- « action réalisée » ;
- « action non possible aujourd’hui » ;
- « journée atypique ».

Temps cible : 3 à 8 secondes.

### Niveau 2 — signature de repas

Une **signature** est un modèle de repas personnel :

- « mon petit-déjeuner semaine » ;
- « salade du midi » ;
- « dîner familial ».

Elle stocke des marqueurs structurés, pas une recette nutritionnelle figée. Le patient modifie seulement les différences.

### Niveau 3 — capture assistée

- dictée ;
- photo ;
- recherche d’un aliment ;
- scan produit ultérieur.

Le système propose des marqueurs. Le patient confirme ou corrige. L’actif brut est supprimé par défaut après validation.

## 4. Empreinte de repas

Chaque repas devient une **empreinte**, non une fiche calorique.

Exemple :

```text
Déjeuner
✓ légumes
✓ légumineuses
✓ huile d’olive
✓ céréales complètes
○ produit ultra-transformé
```

L’empreinte sert à :

- copier un repas ;
- comparer deux moments ;
- repérer une habitude ;
- calculer des fréquences ;
- soutenir un « repas miroir » côté praticien.

## 5. Saisie adaptative

Les questions apparaissent seulement lorsqu’elles changent l’interprétation.

Exemples :

- si « matière grasse » est sélectionnée : demander le type ;
- si « poisson » est sélectionné : gras/autre ;
- si l’action porte sur le petit-déjeuner : ne pas détailler le dîner ;
- si une information a été stable six fois : proposer « comme d’habitude » ;
- si la précision est suffisante : ne plus demander ce marqueur cette semaine.

## 6. Pourquoi maintenant

Toute sollicitation doit contenir une raison concrète :

- « Vous êtes au troisième jour d’observation panoramique. »
- « Cette semaine, il manque encore une occasion de week-end pour comparer. »
- « Votre action concerne le dîner ; une trace après le dîner suffit. »
- « La phase se termine dans deux jours ; cette dernière observation aidera la relecture. »

Jamais : « Vous avez oublié votre journal. »

## 7. Mode semaine difficile

Le patient peut activer le plan minimal sans justification détaillée.

```text
Cette semaine est plus difficile

Votre plan minimal :
noter seulement si l’action principale a été possible aujourd’hui.

[Activer pour 3 jours]
```

Conséquences :

- moins de questions ;
- pas de rattrapage demandé ;
- données marquées comme observation minimale ;
- la météo d’adhésion praticien peut devenir « fragile » avec cause observable ;
- aucun message négatif côté patient.

## 8. Reprise en douceur

Après une interruption :

```text
On reprend ici
Vous n’avez rien à reconstruire.
Souhaitez-vous repartir de votre dernier repère ou saisir aujourd’hui ?
```

Le rattrapage détaillé est limité à 48 heures. Au-delà, l’application propose une reprise au présent.

## 9. Réflexion hebdomadaire

Une seule carte, sans score :

```text
Ce que vos traces rendent visible

• les légumes apparaissent dans la plupart des dîners observés ;
• l’action légumineuses a été possible à deux occasions ;
• les repas du week-end sont encore peu documentés.

Qu’est-ce qui a été le plus facile ?
[Le moment] [La préparation] [L’organisation] [Autre]
```

Le texte est déterministe ou composé à partir d’objets validés. Il ne déduit pas d’effet clinique.

## 10. Transparence et choix

Le centre TRUST doit permettre de comprendre et régler :

- données saisies ;
- finalité ;
- durée de conservation ;
- utilisation de la voix ;
- utilisation de la photo ;
- conservation ou suppression de l’original ;
- traitement par IA ;
- droit de correction, export et retrait.

Photo et dictée sont facultatives et jamais préactivées.

## 11. Accessibilité

- boutons de 44 px minimum ;
- labels toujours textuels ;
- couleur jamais seule ;
- mode confort de lecture compatible ;
- saisie utilisable à une main ;
- aucune interaction basée uniquement sur le glisser-déposer ;
- confirmation avant suppression ;
- messages réseau explicites et reprise hors ligne.
